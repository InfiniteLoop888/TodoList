## 当前功能概览

### 待办与清单

- **多清单**：可创建多个清单、侧栏切换；支持重命名、排序（置顶/上移下移）、删除（至少保留一个清单）。
- **待办项**：勾选完成、内联添加、编辑内容。
- **分区与自动排序**：未完成项始终在上方，已完成项在下方；**新添加的待办默认出现在未完成区最上方**；勾选后自动移入已完成区；取消勾选时，按保存的**手动顺序**回到未完成区中的对应位置。
- **手动排序与存储**：支持**鼠标拖拽**调整顺序，以及右键菜单的置顶/上移/下移；使用 `order_key` 记录清单内的手动顺序，使用 `completed_rank` 记录已完成区内部的显示顺序；旧数据中的 `created_order` 会在读取时兼容并视为 `order_key`；拖拽时目标位置**限制在「未完成」或「已完成」各自分区内**，避免与分区规则冲突。
- **批量操作**：一键**全部完成**当前清单。
- **归档与回收站**：内置归档与回收站系统清单，支持单条归档/移入回收站、一键归档所有已完成项；在归档与回收站视图内可进行「恢复到原清单」与「永久删除」；底部图标支持**右键菜单**一键「全部恢复」或「全部删除」所有归档/回收站项。

### 提醒与日历

- **提醒**：为单条待办设置日期与时间；支持**不重复 / 每天 / 每周 / 每月**，以及**重要程度**（不重要 / 重要 / 紧急）；快捷选项含「30 分钟后」「明天 9 点」「本周日」「下周一」等。
- **到时通知**：后台定时检查，到期弹出提醒窗口（未完成项）。
- **独立日历窗口**：月视图查看带提醒的待办，日期上方附带**悬浮简报弹窗**；顶部包含**悬浮式年月快捷选择弹窗**，支持跨度达十年的快速跳转；可在日历详情视图中直接编辑文字、切换完成状态、重新设置/删除提醒及进行常规操作。

### 窗口与系统集成

- **无边框与系统托盘**：主窗口为无边框工具窗口，通常**不占用任务栏按钮**；支持**底部、右侧、右下角边缘拖拽缩放**调整窗口高宽；托盘菜单可显示窗口、最小化、退出；Windows 下可勾选**开机自启动**（当前用户「运行」注册表项）。
- **外观体验**：支持**深色模式**、**半透明模式**及透明度滑块。提供**防误触锁定模式**，配合半透明效果可实现无感知的“桌面微件”。侧栏具备**平滑的悬浮伸缩动画**；并且得益于强化的**响应式界面重绘机制**，即便侧栏开关或窗口缩放，主视图也能做到严格对齐、不被截断。
- **个性化**：待办正文字号可调（像素，约 8–24）；底部设置面板包含致谢特别样式与彩蛋渲染。

### 配置与数据位置

- 首次运行会在系统<strong>「我的文档」</strong>目录下创建 `TodoList` 文件夹，并将程序目录中的 `options.ini` / `todos.ini` **复制**到该处（若不存在），之后读写均针对用户目录，便于升级程序而不丢数据。
- **数据安全机制**：内置原子化文件写入机制，写入前会自动生成 `.bak` 备份文件，极大程度避免在突然断电或程序崩溃等极端情况下配置文件被意外清空；所有 UI 的增删改操作均与后台文件实时绑定同步，防止内存态与磁盘态数据不一致。

## 如何使用
* 在右侧下载最新的 Release，解压并运行即可
* 如果你需要开机自启，请参照网上的方法将本应用添加到自启


### 从源码运行
在项目根目录安装依赖（需已安装 Python 与 PyQt5 等，与 PyQt-SiliconUI 要求一致）后执行：

```bash
python start.py
```

### 自行打包（PyInstaller）

若官方 Release 无法在你的环境运行，可自行打包，例如：

```bash
pyinstaller start.py --noconsole
```

打包后请将 **`options.ini`、`todos.ini`** 以及 **`icons/icons.dat`** 放到可执行文件同目录（或与你的打包配置一致）；首次启动仍会向「文档/TodoList」同步配置。具体路径以实际报错为准。
